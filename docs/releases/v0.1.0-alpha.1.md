# Bloom v0.1.0-alpha.1

`v0.1.0-alpha.1` is the first public desktop app alpha for Bloom.

This release is intentionally limited to the app shell and visual companion
experience. It is useful for trying the tray, transparent companion window,
dashboard, mock suggestion speech bubble, synthetic activity view, and local
interaction model.

## Included

- Tauri desktop app shell for macOS, Windows, and Linux builds.
- Floating corner companion window with speech bubble suggestion UI.
- Dashboard window with overview, synthetic activity, settings, and platform
  capability views.
- System tray actions for showing the companion, opening the dashboard, pausing
  learning, and quitting.
- In-memory pause and mock suggestion feedback.
- Frontend and Rust unit tests.

## Not Included Yet

Bloom v0.1.0-alpha.1 does not monitor real apps, persist activity, install
terminal hooks, call Ollama, open URLs, launch applications, execute commands,
collect browser history, or run autonomous workflows.

## Privacy

This alpha uses synthetic data only. It does not collect screenshots, screen
recordings, global keystrokes, clipboard history, browser history, terminal
history, private messages, secrets, environment variables, or telemetry.

## Install Notes

These artifacts are unsigned. macOS Gatekeeper and Windows SmartScreen may warn
before opening Bloom. Verify downloads with `SHASUMS256.txt`.

Windows and Linux artifacts are CI-built alpha artifacts. Runtime support claims
remain conservative until those platforms are manually tested.
